Configuración de Telegram

Para recibir recordatorios y avisos, necesitamos tu chat ID personal de Telegram. El teléfono de tu cuenta en Prevy debe ser el mismo que usas en Telegram.

Cómo obtener tu chat ID

  1. Descarga Telegram en tu móvil (App Store o Google Play) o usa Telegram Web.
  2. En el buscador de Telegram busca el bot «Chat Id Info Bot» (o un bot de confianza que te muestre solo tu ID numérico de usuario).
  3. Toca Iniciar / Start en la conversación con ese bot.
  4. El bot te responderá con un número (tu chat ID). Copia solo ese número (sin letras ni espacios).
  5. Vuelve a esta página y pega el número en el formulario de abajo. Guarda. El administrador puede generar en el panel owner un JSON que une la variable TELEGRAM_CHAT_BY_PHONE_JSON con los datos guardados aquí y actualizar Netlify.

Tu chat ID

Si cambias de usuario de Telegram o el bot no te envía mensajes, comprueba que tu teléfono en Mis datos (mandatario) coincida con tu cuenta de Telegram.
